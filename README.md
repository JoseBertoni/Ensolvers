# ToDo-List
Todo list app using
* Spring Boot (v2.6.1)
* Hibernate (Using Spring Data JPA)
* Lombok
* Postgres
* HTML5
* Angular 10
* Bootstrap (v5.0.0-alpha1)  

## Requirements

1. Java - 11

2. Maven - 3.x.x

## Run

Inside the api folder run `mvn spring-boot:run` and visit localhost:8080
